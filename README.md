# Mimodots

Генератор пиксельных иконок. Создавайте псевдопиксельные иконки из SVG-изображений.

## Что делает

- Генерирует пиксельную сетку от 16×16 до 640×480 с любым соотношением сторон
- Импортирует любой SVG — отрисовывается как маска на сетке
- Выбор формы пикселей: квадрат, круг, шестиугольник
- Настройка расстояния между пикселями и выбор любого цвета
- Экспорт чистого векторного SVG с вашей кастомной пиксельной иконкой

## Почему это существует

Потому что английский интерфейс мобильного приложения «Билайн» — это нечто уродское и дурацкое. Каждая иконка выглядит так, будто её нарисовал стажер в пятницу вечером без всякого понимания дизайна. Вот инструмент, чтобы делать нормальные иконки — в стиле пиксель-арт, но на самом деле красивые.

## Как пользоваться

Откройте `index.html` в браузере. Никакой сборки, зависимостей и сервера не нужно.
